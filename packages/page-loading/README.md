页面过渡动画组件。

## Installation

```bash
npm i @antdp/page-loading --save
```

## Basic Usage

<!--rehype:bgWhite=true&codeSandbox=true&codePen=true-->
```jsx
import ReactDOM from 'react-dom';
import PageLoading from '@antdp/page-loading';

const Demo = () => (
  <div>
    <PageLoading />
  </div>
);

ReactDOM.render(<Demo />, _mount_);
```
